GTModelZoo
In this portal you could submit a project and browse through projects by your peers.

To submit a project, you need to sign up. This will allow you to create, update and delete your projects.

Few guidelines:
  • If you have multiple framework(TensorFlow, pyTorch, Caffee etc) implementations for the same task, you could submit them as individual projects or submit the framework with the best performance.

  • Each section of the project submission has a description field.Although it is optional, please provide as much accurate and detailed information as possible. This will help the community to learn.
